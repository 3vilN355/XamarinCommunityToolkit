This project was added to ensure CommunityToolkit.Maui support for F#

Specifically, in an earilier version of CommunityToolkit.Maui, it was reported that F# projects couldn't use the CommunityToolkit.Maui NuGet package: https://github.com/xamarin/XamarinCommunityToolkit/commit/e569ee3a2f4cabaffdcfad11edbc2ef495cbe51b

This project ensures that all future releases of CommunityToolkit.Maui will continue to be compatible with F#.
